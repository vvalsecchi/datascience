Scaricare la cartella denominata covid
Assicurarsi di avere i vari dataset in formato csv nella cartella contenente il file Analisi_COVID-19.ipynb e il file funzioni.ipynb
Analisi e considerazioni sono presenti nel notebook Analisi_COVID-19

Dataset utilizzati:
italy_total #dataset contenente le informazioni sulla situazione italiana totale
italy_region #dataset contenente le informazioni sulla situazione italiana regionale
italy_province #dataset contenente le informazioni sulla situazione italiana provinciale
globale2 #dataset contenente le informaizioni sulla situazione globale
ChIta #dataset generato da noi contenente i dati demografici ed economici delle singole regioni di Italia e Cina. Più in dettaglio presenta lo stato di appartenenza della regione (Italia o Cina), la popolazione, la densità (espressa in ab/km²) e il Prodotto Interno Lordo - PIL - (espresso in miliardi con valuta dollaro statunitense).
chi #dataset generato a partire dai dati contenuti in globale2 per isolare i dati cinesi
